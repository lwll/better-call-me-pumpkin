# 组件

# Chrome插件

TIP

github上有一个chrome插件英雄榜， 推荐了很多优质的chrome插件，更新频率也很快。这里就只记录自己用过的了。

  • 谷歌访问助手 要下chrome插件，首先得翻墙吧？
  • Adblock Plus - 免费的广告拦截器 插件如其名
  • Restlet Client - REST API Testing 非常好用的api测试工具
  • OneTab Plus:标签效率管理扩展 还在为标签页过多又舍不得关掉而烦恼吗？一个插件解决你的问题
  • Infinity 新标签页(Pro) 极大地提高新标签页的作用，又不会显得页面很杂乱
  • Tampermonkey chrome插件唯一真神
  • Octotree github上看代码必备
  • Save to Pocket 将你在浏览器上看到的内容保存在pocket中！页面整洁好看，缺点就是网站打开有点慢。

# 其他

  • ScreenToGif 轻量、好用的录制Gif、视频的工具
  • Everything 搜索文件神器
  • cmder windows下强大的命令行工具
  • Typora 免费、简洁的本地Markdown编辑器
